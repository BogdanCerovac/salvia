Welcome, this is a general accessibility statement for the Salvia web accessibility testing service

This accessibility statement concerns the service Salvia web accessibility testing service and it has been drafted / updated on 26.9.2022. The service is governed by the Act on the Provision of Digital Services, which requires public online services to be accessible.

We have assessed the accessibility of the service ourselves.

Accessibility status of the digital service

Partially meets the accessibility requirements.

Non-accessible content

The website is not yet fully compliant with the requirements

  • Perceivable: links in the PDF-files are not connected with the tag structure.

    Non-accessible content and its shortcomings

    In PDFs clickable links are floting annotations placed above the corresponding links in the text. To meet the accessibility requirements these link annotations need to be connected with the sorresponding LINK tags adding a LINK-OBJR tags to these LINK tags. Missing this connection between the clickable link and the describing link tag might cause some accessibility issues with some PDF readers.
    The program library that's being used to print PDF files, PDFkit, does not yet support this. Therefore, currently all URL addresses have been printed out to the document text.

    Accessibility requirements that have not beet met
    • 1.3.1 Info and Relationships

  • Operable: PDF files lack the information about appropriate reading order

    Non-accessible content and its shortcomings

    The intended reading order of the PDF files is the tag order in the documents' structure. However, the program library that's being used to print PDF files, PDFkit, does not yet support adding this information to the document automatically.

Supervisory authority

If you notice any accessibility issues on the site, please provide feedback to us first, that is, to the site administrator. We will reply within 14 days. If you are not satisfied with our response or do not receive any response within two weeks, you can file a report with the Regional State Administrative Agency for Southern Finland. The page of the Regional State Administrative Agency for Southern Finland contains a detailed description of how the report can be filed and how the matter will be processed.
